# For Example

Super simple example to explain how "for" command works (in Polish).

Prosty przykład do tłumaczenia komendy 'for' w JavaScript. Końcowy efekt:
![Po lewe stronie znajduje się kod w JavaScript odpowiedzialny za wyświetlenie 8 losowych numerów oraz 5 potworow. Po prawej - żółte piłeczki z wylosowanymi numerami oraz 5 wylosowanych postaci potworów](https://user-images.githubusercontent.com/35850828/158964733-d8fca55d-9863-470b-885a-fdb142b73613.png)

Uczestnicy powinni zacząć z brancha step-0 i dopisać brakujący fragment kodu (wzorując się na funkcji `wylosujNumeryNaLoterie`) w taki sposób, aby wyświetlić również 5 losowych potworów.
